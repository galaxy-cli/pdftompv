# SETUP
* Use `chmod u+x pdftompv.sh` to make the script executable
* Use `./pdftompv.sh` to run script

# USAGE
> `pdftompv --pdf [PDF]`
> `pdftompv --open [PDF] [MP3]`
> `pdftompv --url [URL]`

# FLAGS
*  `--pdf`   Convert PDF to MP3
*  `--open`  Open PDF and play MP3
*  `--url`   Download PDF from URL
